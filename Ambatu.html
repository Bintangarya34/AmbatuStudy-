Persamaan Kuadrat   (Contoh + Pembahasan)

Ringkasan cepat — definisi, rumus, dan contoh soal yang disusun rapi untuk belajar mandiri.

Definisi

Persamaan kuadrat adalah persamaan derajat dua dengan bentuk umum \(ax^2 + bx + c = 0\), di mana \(a \neq 0\). Akar-akar dapat dicari dengan rumus kuadrat:

Rumus Kuadrat:
\[x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}\]

Contoh

Selidiki akar persamaan \(2x^2 - 4x - 6 = 0\).

// langkah penyelesaian singkat
// a = 2, b = -4, c = -6
// diskriminan D = b^2 - 4ac = (-4)^2 - 4*2*(-6) = 16 + 48 = 64
// x = (4 ± 8) / 4 => x1 = 3, x2 = -1

Tip belajar

  • Selalu hitung diskriminan dulu untuk mengetahui jumlah akar.
  • Jika \(D > 0\) => 2 akar real; \(D = 0\) => 1 akar (double); \(D < 0\) => akar kompleks.

Latihan singkat

Selesaikan: \(x^2 + 2x + 5 = 0\). (jawaban di akhir halaman)
